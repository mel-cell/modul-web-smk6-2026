Sebelum mulai masuk ke dalam kode **Payload**, ada 3 pilar utama yang harus kamu pahami agar tidak bingung saat membaca struktur project.

## 1. Type Safety dengan TypeScript
Project ini menggunakan **TypeScript**. Kenapa? Agar kita meminimalisir error saat runtime.
- **Tip:** Selalu perhatikan *Interface* atau *Type* yang sudah ada. 
- Jangan menggunakan `any` kecuali benar-benar terpaksa.

## 2. Git Workflow (Ijo-Ijo GitHub)
Kita menggunakan Git bukan cuma untuk simpan kode, tapi untuk kolaborasi.
- **Satu Fitur = Satu Commit:** Pastikan pesan commit kamu jelas (contoh: `feat: add news collection schema`).
- **Ijo-Ijo:** Grafik kontribusi kamu akan bertambah setiap kali kamu melakukan `push` ke repositori dokumentasi ini.

## 3. Terminal Power
Hampir semua kontrol project dilakukan via Terminal. Pastikan kamu terbiasa dengan:
```bash
# Menjalankan project secara lokal
npm run dev

# Membangun docker container
docker-compose up --build
```
