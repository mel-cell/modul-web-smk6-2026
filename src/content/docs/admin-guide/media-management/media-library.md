Semua file yang muncul di website (Foto Berita, Logo Jurusan, PDF Pengumuman) harus melewati **Gudang Media**.

:::note[Dua Jenis Media]
Sistem ini memiliki **dua gudang media** yang terpisah:
1. **Media** — Untuk gambar umum (berita, galeri, logo, dll).
2. **Koleksi Media 360** — Khusus untuk foto panorama 360° (Smart Tour). Lihat panduan terpisah di bagian Smart Tour.
:::

## 1. Akses Pustaka Media

Klik menu **Media** pada bagian atas Sidebar. Anda akan melihat semua file yang pernah diunggah ke website.

## 2. Cara Unggah (Upload)

1. Klik tombol **Create New**.
2. Pilih file dari komputer Anda (Drag & Drop juga bisa).
3. Isi kolom **Alt Text** (Teks Alternatif). 
   *   *Penting:* Alt Text membantu orang dengan keterbatasan netra dan membantu Google menemukan gambar sekolah Anda.

### Format & Ukuran Otomatis
Sistem akan **otomatis mengkonversi** gambar ke format **WebP** dan membuat 7 ukuran berbeda (thumbnail, square, small, medium, large, xlarge, og). Anda tidak perlu menyiapkan berbagai ukuran gambar secara manual.

## 3. Fitur Galeri Publik

Setiap gambar memiliki opsi **"Tampilkan di Galeri Publik"** di sidebar kanan. Jika dicentang:
-   Gambar akan otomatis muncul di halaman **Galeri** website.
-   Anda bisa memilih **Kategori Galeri** (contoh: Acara, Fasilitas, Kegiatan).

:::tip[Tips]
Gunakan fitur ini untuk membuat galeri foto kegiatan sekolah yang rapi tanpa perlu membuat halaman baru.
:::

## 4. Merapikan File dengan Folder

Klik tombol **"Browse by Folder"** di pojok kiri atas halaman Media untuk mengatur file ke dalam folder-folder (misal: "Berita 2026", "Foto Ekskul", "Logo Partner").

## 5. Penghapusan Aset

Hati-hati saat menghapus gambar dari Pustaka Media. Jika gambar tersebut sedang dipakai di sebuah Berita, maka berita tersebut akan menampilkan "Gambar Rusak" (Broken Link).

:::warning[Tips Hemat Ruang]
Gunakan nama file yang jelas (misal: `lomba-voli-2024.jpg` bukan `IMG_1234.jpg`) agar mudah dicari di kemudian hari. Ukuran file yang disarankan: **maksimal 2 MB** untuk foto biasa.
:::
