Project SMK6 Engine menggunakan teknologi **Page Builder**. Artinya, Anda bisa menyusun layout halaman seperti menyusun Balok mainan.

## 1. Menuju Editor Halaman
Buka menu **CMS General** > **Halaman** > Pilih halaman yang ingin diubah (Misal: Home).

:::tip[Area Gambar: List Halaman]
![Koleksi Halaman CMS](../../../../../../assets/admin-guide/content-management/list_halaman.png)
:::

## 2. Struktur Balok (Layout Blocks)
Halaman web kita terdiri dari tumpukan Balok. Setiap balok punya tugas berbeda:
*   **Balok Hero**: Banner selamat datang.
*   **Balok Berita**: Menampilkan list berita secara otomatis.
*   **Balok Galeri**: Barisan foto-foto kegiatan.

## 3. Kekuatan Page Builder
*   **Hapus**: Ingin menghilangkan satu bagian? Klik ikon sampah.
*   **Geser**: Ingin menaikkan posisi pengumuman? Gunakan fitur Drag & Drop di sisi kiri balok.
*   **Ganti Isi**: Edit isian di dalam balok tanpa merusak balok lainnya.
