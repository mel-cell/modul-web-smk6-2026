Setiap Kompetensi Keahlian (Jurusan) memiliki halaman mading digitalnya sendiri. Di sini Anda bisa memamerkan keunggulan jurusan masing-masing.

:::tip[Area Gambar: Dashboard Jurusan]
![Daftar Jurusan Sekolah](https://via.placeholder.com/1200x500?text=Screenshot+Daftar+Jurusan)
:::

## Menu Utama Jurusan
1. **Daftar Jurusan**: Melihat semua kompetensi keahlian yang aktif.
2. **Tambah Jurusan**: (Hanya untuk Admin Utama) jika ada pembukaan jurusan baru.
3. **Edit Jurusan**: Klik pada nama jurusan untuk mengubah informasi detailnya.

:::note
Informasi Jurusan adalah hal pertama yang dicari oleh calon siswa baru (PPDB). Pastikan data ini selalu segar dan menarik.
:::
