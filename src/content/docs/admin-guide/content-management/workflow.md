Saat Anda mengedit data, ada beberapa tombol penting di bagian bawah layar. Memahami tombol ini sangat penting agar perubahan Anda tidak hilang.

:::tip[Area Gambar: Tombol Publish/Save]
![Screenshot Tombol Publish](https://via.placeholder.com/600x200?text=Upload+Screenshot+Tombol+Save+vs+Publish)
*Gunakan gambar yang fokus pada area tombol di pojok kanan bawah atau bawah layar.*
:::

## 1. Simpan sebagai Draft (`Save Draft`)

Gunakan tombol ini jika tulisan Anda belum selesai. 
*   **Efek:** Data tersimpan di sistem, tapi **TIDAK MUNCUL** di website publik (Siswa/Tamu belum bisa baca).
*   **Kegunaan:** Sangat baik untuk menyiapkan berita yang masih butuh revisi.

## 2. Terbitkan (`Publish`)

Gunakan tombol ini jika data sudah benar-benar siap.
*   **Efek:** Data langsung dikirim ke website dan **BISA DILIHAT** oleh semua orang.
*   **Penting**: Pastikan foto dan ejaan sudah benar sebelum menekan tombol ini.

## 3. Menghapus Data (`Delete`)

Jika sebuah data sudah tidak relevan lagi (misal berita tahun lalu yang salah tulis), Anda bisa menghapusnya secara permanen.

:::danger[Waspada]
Data yang sudah dihapus secara permanen **TIDAK DAPAT DIKEMBALIKAN**. Pastikan Anda benar-benar yakin sebelum menekan tombol Hapus.
:::

## 4. Cara Update (Ubah) Data

1. Masuk ke daftar koleksi (misal Berita).
2. Cari data yang ingin diubah, klik pada judulnya.
3. Ubah isinya.
4. Tekan tombol **Save Change** atau **Publish** ulang.
