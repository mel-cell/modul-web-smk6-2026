Page Blocks adalah "batu bata" yang digunakan untuk **menyusun halaman website**. Admin bisa menambahkan, menghapus, dan mengatur urutan blok di setiap halaman melalui Admin Panel.

## Cara Menggunakan Blok

1.  Buka halaman yang ingin diedit di menu **Halaman**.
2.  Scroll ke bagian **"Content Blocks"** atau **"Layout"**.
3.  Klik tombol **"Add Block"**.
4.  Pilih blok yang diinginkan dari daftar.
5.  Isi konten blok tersebut, lalu **Save**.

## Daftar Blok Lengkap

|------|--------|-------------------|

:::tip[Tips Menyusun Halaman]
1.  **Mulai dengan Banner** untuk tampilan pertama yang memukau.
2.  **Selang-seling** antara blok teks dan visual agar tidak monoton.
3.  **Akhiri dengan CTA** agar pengunjung tahu langkah selanjutnya.
4.  **Gunakan Spacer** untuk memberikan "ruang bernapas" antar bagian.
:::
